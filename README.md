# Todo Pages

依存なしでそのまま GitHub に置ける、複数ページ対応の手帳風 TODO アプリです。

## 機能

- 紙全体がめくれるページ切り替え
- ページごとの折れ角、色付き付箋タブ、インク色切り替え
- ページ追加時に奥の紙束が増える演出
- 手動順では長押しで並び替え
- 四角を押すとチェック、完了時に朱肉風の `済` スタンプ
- `急ぎ` `今日` `買う` の丸スタンプを項目ごとに設定
- 文字を押すと紙の裏面風メモが開き、メモ、期限、スタンプ、ピン留めを編集
- 右下の `AI分析` から Noesia Persona API に現在ページの内容を渡し、固定の見方ボタンで人格差のあるコメントをもらえる
- ピン留めした項目は上に固定、クリップ風表示
- 完了項目の一括削除とページ削除に破る演出
- データは `localStorage` に保存

## Noesia API

- 右下の `AI分析` を押す
- 右上の `API設定` から、表示名と `Noesia の sk_ APIキー` を追加する
- 追加した API を `使う` で選ぶ
- `ひとこと感想` `気になる点` `後回し警報` `持ち主にひとこと` `抜けチェック` の固定ボタンを押す
- 同じ TODO でも人格ごとに反応や指摘が変わるか見比べられる
- 接続先は `https://noesia.onrender.com` を前提にしている
- エラー時は `request_id` 付きで表示されることがある

## 公開

`index.html`、`styles.css`、`app.js` をそのまま GitHub Pages に置けば動きます。
